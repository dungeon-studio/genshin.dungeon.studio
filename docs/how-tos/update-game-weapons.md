# How to regenerate the Genshin Impact weapon roster

`packages/game-data/src/weapons.generated.ts` is generated by
`@genshin/game-data-codegen`. Edit the generator, not the file.

## Regenerate the roster

Optionally pull newer game data first:

```bash
pnpm --filter @genshin/game-data-codegen up genshin-db
```

Build the generator and run it:

```bash
pnpm turbo run build --filter @genshin/game-data-codegen
pnpm --filter @genshin/game-data-codegen generate weapons
```

Confirm the gates pass:

```bash
pnpm turbo run typecheck test lint --filter @genshin/game-data
```

## Add a sub-stat the generator rejects

`generate weapons` aborts when `genshin-db` reports a `FIGHT_PROP_*` sub-stat
with no mapping. To add one:

1. Add the member to `WEAPON_STAT_TYPES` in `packages/game-data/src/weapons.ts`.
2. Map the `FIGHT_PROP_*` name to it in `SUB_STAT_BY_GENSHIN_DB` in
   `tools/game-data-codegen/src/weapons.ts`.
3. Regenerate the roster.
